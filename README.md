## Description

Examples of how to create pipes in nestjs


## Project setup

```bash
  # install packages
  npm install
```

## Compile and run the project

```bash
  # watch mode
  npm run start:dev
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
